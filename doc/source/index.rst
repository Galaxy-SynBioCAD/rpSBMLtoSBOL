Welcome to rpSBMLtoSBOL's documentation. This tool provides a docker that converts rpSBML_ files to SBOL_ files. The project uses the UNIPROT id's generated by rpSelenzyme_ to fetch the coding sequences of the heterologous reaction that are generated by RetroPath2.0_. Then these are used to generate the SBOL_.

.. code-block:: bash

   docker build -t brsynth/rpsbmltosbml-standalone .

You can run the docker using the following command:

.. code-block:: bash

   python run.py -input test/test_rpGlobalScore.tar  -input_format tar -output test/test_sbol.tar
